# PRD — MVP Live (bpslib.dk)

## Formål
Lancér et nyt, hurtigt og SEO-sikkert MVP-site for bpslib.dk, hvor bordplader er primært fokus. Vi overtager gamle Base44-URL’er med 301 fra dag 1, selvom Base44 slettes.

## Mål og succes
- MVP live med kerneindhold, prisflow og lead-gen.
- Ingen SEO-tab: korrekt canonical, redirects, sitemap og interne links.
- Bordplade (massiv træ) er primært fokus. Øvrige fag håndteres kun som lead-gen.

## Ikke-forhandlingsbare krav
- Alt indhold er på dansk.
- URL-standard: lowercase + bindestreger + ingen trailing slash.
- Canonical peger altid på lowercase uden trailing slash.
- Redirect-lag i den nye app håndterer gamle Base44-paths med 301.
- Prisberegner og booking: kun massiv træ.
- Sekundære fag: lead-gen via `/tilbudstid` (ingen online booking i MVP).

## MVP scope (skal med ved launch)
### Forside
- Klar value proposition for bordpladeslibning.
- Primær CTA til prisberegner/booking.
- Sekundær CTA til tilbudstid (lead-gen).

### Bordplade pillar (primær service)
- Én samlet service-side for bordplader i massiv træ.
- Understøtter pris, proces og forventningsafstemning.

### Pris + Prisberegner (MVP)
- Pris-side med tydelige prisrammer og forklaringer.
- Prisberegner med filupload (billeder) og manuel vurdering.
- Ingen automatisk pris i MVP.

### Booking (slot-baseret)
- Visning af ledige tider.
- Booking kun for massiv træ.
- Bekræftelse via mail.

### Akutte tider
- 14-dages fast pris.
- Tydelig kort tekst og CTA.

### Tilbudstid (lead-gen)
- Formular til andre fag end bordplader.
- Ingen online booking i MVP.

### Cases (simpel)
- Enkelt grid med billeder og korte tekster.
- Fokus på bordplader.

### Om os
- Kort historik, metode og kvalitet.

### Kontakt
- Telefon, mail, åbningstider, adresse.

### Privatliv, Cookie, Handelsbetingelser
- Standard juridiske sider i MVP-format.

## Indholds- og UX-krav
- Desktop er primær.
- Tydelige, korte CTA’er.
- Formularer med enkel validering og kvittering.
- Alle UI-tekster på dansk.

## Efter launch (ikke i MVP)
- Intent-sider (f.eks. “bordpladeslibning pris”, “oliebehandling bordplade”).
- Lokationssider kun hvis der findes unikke cases pr. lokation.
- Anmeldelser-side (Google/Trustpilot).
- Guides og vidensunivers.
- Udvidet automatisk prisberegning.
